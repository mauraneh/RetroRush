
Importer :
    import Alert  from '../components/Alert';
    const [alert, setAlert] = useState({ show: false, type: '', message: '' });

Défaite :
        setAlert({ show: true, type: 'error', message: 'Vous avez perdu ! ' });

Victoire :
        setAlert({ show: true, type: 'win', message: 'Bravo, Vous avez gané ! ' });

Appel dans le return :
        {alert.show &&
            <Alert status={alert.type} message={alert.message} onRestart={initializeGame} show={alert.show} />
        }
onRestart est la méthode de lancement de la partie dans les minijeux ( en tout cas le mien :D )

Bien penser, dans la méthode de lancement de la partie ( ex: initializeGame dans le BreakOut)
    cacher l'alerte :         setAlert({ show: false, type: '', message: '' });
